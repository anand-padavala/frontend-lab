The Evolution of HTML

From plain text documents in 1991 to rich interactive applications in 2014. Click any era to see the tags introduced that year — with live examples.

1991
HTML — The Beginning
Tim Berners-Lee invents HTML at CERN. Just 18 tags. Text, headings, lists, and the revolutionary <a> link tag. No images. No colors. No forms. Digital paper with clickable words.
<h1> <p> <a> <ul> <ol> <li> <dl> <dt> <dd> <address>

1993
HTML+ — Images Arrive
Marc Andreessen (age 22) proposes the <img> tag in an email. Doesn't wait for approval — ships it in Mosaic browser. The web goes from text-only to visual overnight.
<img> <br> <hr>
View 1993 Tags →
1995
HTML 2.0 — Forms & Tables
The first official HTML standard. Users can now type into web pages with forms. Tables allow structured data. The web becomes interactive.
<form> <input> <select> <textarea> <table> <tr> <td>
View 1995 Tags →
1997
HTML 3.2 — Color & Style
The web gets colorful. Background colors, font faces, sizes, and text alignment. Styling was done directly in HTML (CSS was brand new). The era of <font> and <center> tags.
<font> <center> <b> <i> <u> <bgcolor> <sup> <sub>
View 1997 Tags →
1999
HTML 4.01 — Structure & Scripting
HTML matures. <div> and <span> enable CSS layouts. <script> embeds JavaScript. <style> embeds CSS. The <font> tag is deprecated — "use CSS instead."
<div> <span> <script> <style> <iframe> <label> <fieldset>
View 1999 Tags →
2014
HTML5 — The Modern Web
The biggest update ever. Native video and audio (no Flash). Semantic tags for page structure. Canvas for drawing. New form inputs. The web becomes an application platform.
<video> <audio> <canvas> <header> <footer> <nav> <article> <section>

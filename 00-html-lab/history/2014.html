1. Semantic Tags — Meaningful Structure

Before HTML5, everything was <div>. A page had divs for header, divs for navigation, divs for content. The browser had no idea what each div meant. 

  Before (HTML 4):                  After (HTML5):

  <div id="header">               <header>
  <div id="nav">                  <nav>
  <div id="main">                 <main>
  <div id="sidebar">              <aside>
  <div class="post">              <article>
  <div id="footer">               <footer>

HTML5 semantic tags give meaning to sections. Screen readers, search engines, and browsers all understand what each part is.

2. Video — <video> — Native Video Playback

Before HTML5, playing video on the web required Adobe Flash — a browser plugin. Flash was slow, insecure, and didn't work on iPhones (Steve Jobs refused to support it). HTML5 made video native — no plugins needed.

A sample HTML5 video with built-in controls (play, pause, volume, fullscreen).

  <video width="640" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support video.
  </video>

  controls   = show play/pause/volume buttons
  autoplay   = start playing automatically
  loop       = repeat when finished
  muted      = start muted
  poster     = show an image before video plays

3. Audio — <audio> — Native Audio Playback

Same idea as video — play audio without plugins:

A sample HTML5 audio player.

  <audio controls>
    <source src="song.mp3" type="audio/mpeg">
    Your browser does not support audio.
  </audio>

4. Canvas — <canvas> — Draw with JavaScript

The canvas tag gives you a blank drawing area. You draw on it using JavaScript — shapes, text, images, animations, even games.

Shapes drawn with JavaScript on a <canvas> element.

  <canvas id="myCanvas" width="500" height="200"></canvas>

  <script>
    var c = document.getElementById("myCanvas");
    var ctx = c.getContext("2d");

    ctx.fillStyle = "blue";
    ctx.fillRect(20, 20, 120, 80);     // rectangle

    ctx.beginPath();
    ctx.arc(200, 60, 50, 0, 2 * Math.PI);  // circle
    ctx.fillStyle = "red";
    ctx.fill();
  </script>

What is HTML5?

HTML5 is the fifth major version of HTML, published in 2014. It added semantic elements, multimedia support, canvas drawing, new form inputs, and many APIs for building web applications.

Why was Flash replaced?

Adobe Flash was a browser plugin needed for video, audio, and games. It was slow, had security vulnerabilities, and didn't work on mobile devices. Steve Jobs refused to support it on iPhones in 2010, accelerating its decline. HTML5's native <video> and <canvas> tags made Flash unnecessary. Flash was officially discontinued in December 2020.

What came after HTML5?

HTML is now a "living standard" — there is no HTML6. The WHATWG (Web Hypertext Application Technology Working Group) continuously updates the HTML specification. New features are added gradually rather than in big version jumps.

  <details>
    <summary>Click to expand</summary>
    <p>Hidden content revealed when clicked.</p>
  </details>

Meter — Display a measurement

Disk usage: 7 out of 10 GB 7/10 GB
Battery: 30% 30%
Score: 85% 85/100 

  <meter value="7" min="0" max="10">7/10</meter>

Progress — Show task completion

Download: 65%
Loading: (indeterminate — no value set) 

  <progress value="65" max="100"></progress>
  <progress></progress>   <!-- indeterminate -->

7. New Text-Level Tags

Mark — highlighted text (like a yellow marker):
<mark>highlighted</mark>

Time — machine-readable date/time:
Published on October 28, 2014
<time datetime="2014-10-28">October 28, 2014</time>

Output — result of a calculation:
3 + 4 = 7
<output>7</output>

Figure & Figcaption — image with caption: 

  <figure>
    <img src="photo.jpg" alt="Description">
    <figcaption>Caption text</figcaption>
  </figure>

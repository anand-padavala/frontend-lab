HTML — 1991

Tim Berners-Lee, CERN, Switzerland

This page uses ONLY tags available in the original 1991 HTML. No images. No tables. No forms. No colors. No CSS. No JavaScript. Just text, headings, links, and lists — exactly how the first web pages looked.

The 18 Original Tags

HTML started with just 18 tags. Every tag here was part of Tim Berners-Lee's original specification. This is the entire language — nothing more existed.

1. Title — <title>

The title tag sets the text shown in the browser's title bar. Look at the top of this window — it says "HTML 1991 — The Beginning". That is the title tag at work. Every page needs one. 

  <title>My Page Title</title>

2. Headings — <h1> to <h6>

Six levels of headings, from largest to smallest:

Heading Level 1 — Main title

Heading Level 2 — Section

Heading Level 3 — Sub-section

Heading Level 4 — Detail

Heading Level 5 — Minor detail
Heading Level 6 — Smallest heading
  <h1>Heading Level 1</h1>
  <h2>Heading Level 2</h2>
  <h3>Heading Level 3</h3>

3. Paragraph — <p>

The paragraph tag wraps a block of text. Without it, all text runs together on one line.

This is a second paragraph. Notice the gap between paragraphs. The browser adds spacing automatically. 

  <p>First paragraph.</p>
  <p>Second paragraph.</p>

4. Anchor (Link) — <a href=""> — THE Invention

This is the tag that made the web THE WEB. Before this, documents were isolated files. This tag connects one page to another — creating a web of documents.

Go back to the Timeline

Here is a link to an external site:

The First Website Ever (CERN, 1991) 
  <a href="other-page.html">Click here</a>
  <a href="http://info.cern.ch">External link</a>

5. Unordered List — <ul> <li>

A bulleted list:

  • HTML was created at CERN
  • It was designed for sharing research papers
  • The first browser was also a text editor
  <ul>
    <li>Item one</li>
    <li>Item two</li>
  </ul>

6. Ordered List — <ol> <li>

A numbered list:

  1. Write HTML document
  2. Save as .html file
  3. Open in browser
  4. Share the URL
  <ol>
    <li>Step one</li>
    <li>Step two</li>
  </ol>

7. Definition List — <dl> <dt> <dd>

Used for term-definition pairs (like a glossary):

HTML
HyperText Markup Language — the language of web pages
HTTP
HyperText Transfer Protocol — how browsers talk to servers
URL
Uniform Resource Locator — the address of a web page
CERN
European Organization for Nuclear Research — where HTML was born
  <dl>
    <dt>Term</dt>
    <dd>Definition of the term</dd>
  </dl>

8. Address — <address>

Used to show contact information for the page author:

Tim Berners-Lee, CERN, Geneva, Switzerland 
  <address>Author Name, Organization</address>

9. Preformatted Text — <pre>

Displays text exactly as typed — preserving spaces and line breaks. All code examples on this page use the pre tag. 

  This    text    has    spaces.
  And this is on a new line.
     This is indented.

10. Horizontal Rule — <hr>

Draws a horizontal line to separate sections (used throughout this page):

  <hr>

What 1991 HTML Could NOT Do

  • No images (added 1993)
  • No tables (added 1995)
  • No forms or user input (added 1995)
  • No colors or fonts (added 1997)
  • No CSS styling (added 1996)
  • No JavaScript (added 1995)
  • No video or audio (added 2014)

This page is exactly what the web looked like in 1991 — plain text on a white background with blue links. Every website in the world looked like this.

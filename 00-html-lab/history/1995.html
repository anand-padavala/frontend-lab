HTML 2.0 — 1995

The First Official Standard — Forms & Tables

HTML 2.0 was the first formally standardized version of HTML. The two biggest additions: forms (users can type into web pages) and tables (structured rows and columns).

Before this, the web was read-only. After this, users could send data to servers — search queries, login credentials, orders.

New Tags Introduced in HTML 2.0

1. Form — <form>

The form tag wraps input fields and sends data to a server. This is what made e-commerce, search engines, and logins possible.

Your Name:

Your Email:

  <form action="/submit" method="POST">
    <input type="text" name="username">
    <input type="submit" value="Submit">
  </form>

  action = where to send the data (URL)
  method = how to send it (GET or POST)

  <input type="radio">      Radio button (pick one)
  <input type="hidden">     Invisible data
  <input type="submit">     Submit button
  <input type="reset">      Clear all fields

3. Select (Dropdown) — <select> <option>

A dropdown menu for choosing from a list:

Choose your country:

Select multiple (hold Ctrl to pick more than one):

  <select name="country">
    <option value="in">India</option>
    <option value="us">United States</option>
  </select>

4. Textarea — <textarea>

A multi-line text input (for comments, messages, etc.):

Your Message:

  <textarea name="message" rows="5" cols="40">
    Default text here
  </textarea>

  rows = height (number of visible lines)
  cols = width (number of characters per line)

5. Table — <table> <tr> <th> <td>

Display data in rows and columns:

Name City Language
Anand Hyderabad Telugu
Ravi Chennai Tamil
Sita Mumbai Marathi 
  <table border="1">
    <tr>                       ← table row
      <th>Name</th>           ← table header (bold)
      <th>City</th>
    </tr>
    <tr>
      <td>Anand</td>          ← table data (normal)
      <td>Hyderabad</td>
    </tr>
  </table>

6. Caption — <caption>

A title for a table:

Top 3 Programming Languages (1995)
Rank Language Created
1 C 1972
2 C++ 1983
3 Java 1995 
  <table>
    <caption>Table Title</caption>
    ...
  </table>

Note: In 1995, tables were used for LAYOUT — not just data. Every website used tables to position elements on the page. This lasted until CSS became mature around 2003–2005.

Tags Summary

  <form>          Wraps all inputs, sends data to server
  <input>         Text, password, checkbox, radio, submit, reset
  <select>        Dropdown menu
  <option>        Item in a dropdown
  <textarea>      Multi-line text input
  <table>         Table container
  <tr>            Table row
  <th>            Table header cell (bold)
  <td>            Table data cell
  <caption>       Table title

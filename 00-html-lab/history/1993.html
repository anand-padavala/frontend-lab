HTML+ — 1993

Marc Andreessen adds <img> to Mosaic browser

This page demonstrates the tags introduced around 1993. The biggest one: the <img> tag — images on the web!

New Tags Introduced in 1993

1. Image — <img> — The Game Changer

Before 1993, the web was text-only. Marc Andreessen, a 22-year-old developer building the Mosaic browser, sent an email proposing: 

  "I'd like to propose a new optional HTML tag: IMG"

Others debated for weeks — should it be <img>, <icon>, <include>? Meanwhile, Andreessen just shipped it in Mosaic. People used it. The debate became irrelevant.

Here is the img tag in action (using a placeholder image):

The syntax:

  <img src="photo.gif" alt="Description of the image">

  src = source — where the image file is
  alt = alternative text — shown if image fails to load

Note: In 1993, images were GIF or JPEG only. PNG didn't exist until 1996. Photos were tiny — a 50KB image could take 30 seconds to load on a modem.

2. Line Break — <br>

Forces a new line without starting a new paragraph. Useful for addresses or poetry:

Tim Berners-Lee
CERN
Geneva, Switzerland
CH-1211 

  Tim Berners-Lee<br>
  CERN<br>
  Geneva, Switzerland<br>

What Changed

Before 1993
Web = text and links only. Like reading a research paper.
After 1993
Web = text, links, AND images. Like reading a magazine.

This single tag — <img> — transformed the web from an academic tool into something visual and accessible to everyone.

Tags Summary

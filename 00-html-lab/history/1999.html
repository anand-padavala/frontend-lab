HTML 4.01 — 1999

Structure, Scripting & the Birth of Modern Web

HTML 4.01 is where the web grew up. This version introduced <div> and <span> for CSS-based layouts, <style> for embedding CSS, <script> for embedding JavaScript, and deprecated old tags like <font> and <center>.

This page itself uses HTML 4.01 features — notice it has a <style> block in the head. No more inline <font> tags everywhere!

New Tags Introduced in HTML 4.01

1. Div — <div> — The Generic Container

A <div> is a block-level container. It groups content together so you can style the whole group with CSS. This replaced tables for layout.

This is inside a <div>
The div has a border, padding, and white background — all set by CSS class "box". No <table> needed for layout anymore.
Another <div>
Different background color, same class with an override. Divs stack vertically by default. 
  <div class="box">
    Content goes here.
    Style it with CSS instead of HTML attributes.
  </div>

2. Span — <span> — The Inline Container

A <span> is an inline container — it wraps a piece of text within a line so you can style just that piece.

This is normal text with a red word and a blue word and a highlighted phrase in the middle. 

  <p>
    Normal text with a
    <span class="red">red word</span> in the middle.
  </p>

  <!-- Compare: -->
  <div>  = block container (takes full width, stacks vertically)
  <span> = inline container (flows within text)

3. Style — <style> — Embedded CSS

Before HTML 4.01, all styling was done with <font>, bgcolor, etc. Now you could write CSS rules in a <style> block in the <head>. 

  <head>
    <style>
      body {
        font-family: Verdana, sans-serif;
        background: #f5f5f5;
      }
      h1 { color: #003366; }
      .red { color: red; }
    </style>
  </head>

  Before (HTML 3.2):
    <font face="Verdana" color="red">text</font>

  After (HTML 4.01 + CSS):
    <span class="red">text</span>

This was the big shift: separate content (HTML) from presentation (CSS).

4. Script — <script> — Embedded JavaScript

The <script> tag lets you run JavaScript directly in the page. JavaScript was invented in 1995, but it wasn't formally part of HTML until 4.01.

Another demo — current date:

  <script>
    document.getElementById('demo').innerHTML = 'New text!';
  </script>

  <button onclick="alert('Hello!')">Click me</button>

5. Label — <label> — Clickable Form Labels

Clicking a <label> focuses its associated input. Try clicking the word "Username" below — it focuses the text box. 

  <label for="user">Username:</label>
  <input type="text" id="user">

  for="user" links the label to the input with id="user"
  Clicking the label focuses/checks the input.

6. Fieldset & Legend — <fieldset> <legend>

Groups related form fields together with a visual border and title:

    ... form fields here ...
  </fieldset>

7. Iframe — <iframe> — Embed Another Page

An iframe embeds a completely separate web page inside your page. Used for ads, maps, videos, and widgets.

Above: the 1991 page is embedded inside this page using an iframe.

  <iframe src="other-page.html" width="500" height="300">
    Fallback text if browser doesn't support iframes.
  </iframe>

8. Deprecated Tags — What HTML 4.01 Removed

HTML 4.01 officially deprecated these tags — they still worked, but developers were told to stop using them:

Deprecated Tag Replacement
<font> CSS font-family, font-size, color
<center> CSS text-align: center
<u> CSS text-decoration: underline
<s> / <strike> CSS text-decoration: line-through
bgcolor attribute CSS background-color

The message was clear: HTML is for structure, CSS is for styling.

Tags Summary

  <div>          Block container — for CSS layouts
  <span>         Inline container — style text within a line
  <style>        Embed CSS rules in the page
  <script>       Embed JavaScript in the page
  <label>        Clickable form label
  <fieldset>     Group form fields with a border
  <legend>       Title for a fieldset
  <iframe>       Embed another page inside your page

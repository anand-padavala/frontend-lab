HTML 3.2 — 1997

The Web Gets Colorful

This page is styled using ONLY HTML 3.2 tags — no CSS at all. All colors, fonts, sizes, and alignment are done with HTML attributes. This is exactly how websites looked in 1997.

New Tags Introduced in HTML 3.2

1. Font — <font> — Colors, Faces, Sizes

The font tag controls how text looks:

This is Times New Roman, size 5, red

This is Arial, size 4, blue

This is Courier New, size 3, green

This is Georgia, size 6, orange

  <font face="Arial" size="4" color="blue">
    Styled text here
  </font>

  face  = font family (Arial, Times, Courier, etc.)
  size  = 1 (tiny) to 7 (huge), default is 3
  color = color name or hex code (#ff0000)

Font sizes 1 through 7:

Size 1 — tiny
Size 2 — small
Size 3 — normal (default)
Size 4 — medium
Size 5 — large
Size 6 — very large
Size 7 — biggest

2. Center — <center>

Centers everything inside it:

This text is centered.

So is this paragraph.

  <center>
    <p>This is centered.</p>
    <img src="image.gif">
  </center>

3. Bold, Italic, Underline — <b> <i> <u>

This text is bold.
This text is italic.
This text is underlined.
This text is bold AND italic.
This text is bold, italic, AND underlined. 

  <b>Bold</b>
  <i>Italic</i>
  <u>Underline</u>

4. Background Color — bgcolor

The body tag can set a page background color:

  <body bgcolor="#ffffee">   ← this page uses a cream background

  Common colors in 1997:
  bgcolor="white"     #ffffff
  bgcolor="yellow"    #ffff00
  bgcolor="#cccccc"    light gray
  bgcolor="black"     #000000 (with white text)

Table cells can also have background colors:

Red cell Green cell Blue cell
Yellow cell Pink cell Sky blue cell

5. Superscript & Subscript — <sup> <sub>

Used for math and chemistry:

E = mc2 (Einstein's equation)
x2 + y2 = z2 (Pythagorean theorem)
H2O (water)
CO2 (carbon dioxide)
103 = 1000 

  E = mc<sup>2</sup>       Superscript (above)
  H<sub>2</sub>O           Subscript (below)

6. Strikethrough — <s> or <strike>

Regular price: $999 $499!
This feature has been removed. 

  <s>Crossed out text</s>
  <strike>Also crossed out</strike>

7. Blockquote — <blockquote>

Indents a block of text, used for quotations:

"The web is more a social creation than a technical one. I designed it for a social effect — to help people work together — and not as a technical toy."

— Tim Berners-Lee 
  <blockquote>
    Quoted text here. It gets indented.
  </blockquote>

A Typical 1997 Website

Here is what a website looked like in 1997 — built entirely with HTML 3.2 tags:

My Homepage
Welcome to my corner of the internet!

Navigation

Home
About Me
Photos
Links
Guestbook
Welcome!

This is my personal homepage. I like computers, music, and web design.

You are visitor number: 001,342
Last updated: March 15, 1997
Best viewed in Netscape Navigator 3.0 at 800x600
Copyright 1997. Made with Notepad.

Notice: the entire layout is built with tables — a navigation sidebar on the left, content on the right. This was the standard technique until CSS layouts took over around 2003.
